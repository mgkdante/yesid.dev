A: Centered Stack

SVG icon centered top, benefit headline centered, service name as muted label, stack tags at bottom. Symmetric and clean. Reads like a product card.

Pros

Cons

  • Centered text can feel generic
  • Less dynamic than left-aligned

C: Compact Header + Flow

Small SVG icon inline with service label at top. Benefit headline large below. Description. Metric (large, orange, monospace) and stack tags share the bottom row. Most information-dense while staying clean.

Pros

  • Most info per card without feeling crowded
  • Metric gets visual weight (large + orange)
  • SVG stays present without dominating
  • Fits 3-col grid naturally

Cons

  • SVG is smaller (56px) — less visual anchor
  • More elements to balance
