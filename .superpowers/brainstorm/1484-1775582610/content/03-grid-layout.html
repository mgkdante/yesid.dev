Bento Grid — Responsive Layout Strategy

Before designing cards, let's lock the grid. The current layout stretches infinitely on wide screens. Pick a containment strategy.

A

Max-Width Container (centered)

Grid caps at 1400px, centered on wide screens. Cards never get absurdly wide. Content stays readable. Side margins grow on ultrawide. Used by: Vercel, Linear, Stripe.

B

Full Stretch (current)

Grid fills 100% width at all sizes. Dashboard feel. Cards grow with viewport. Risk: on 2560px monitors, cards become very wide and text floats left. Used by: Notion, some dashboards.

C

Fluid + Max (hybrid)

Full width up to 1600px, then centered with generous side padding. Best of both — feels expansive on normal screens, doesn't break on ultrawide. Used by: Apple, Awwwards sites.
