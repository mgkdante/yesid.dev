cursorGlow Action

Svelte action (use:cursorGlow) tracks pointermove and updates --glow-x, --glow-y CSS custom properties. The existing static radial-gradient on BlogRow and WorkCard follows the cursor.

ME-3

Animated Gradient Border

Pure CSS using @property --angle + conic-gradient. The border rotates on hover, paused by default. Replaces the static border-color transition on WorkCard.

@property --angle: 0deg → 360deg
conic-gradient(from var(--angle), ...)
animation-play-state: paused → running
