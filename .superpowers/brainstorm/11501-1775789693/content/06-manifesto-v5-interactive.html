INTERACTIVE ELEMENTS — WHAT'S NEW

NODE PROXIMITY GLOW

Circuit nodes within 120px of cursor brighten from 15% to 60% opacity and gain an orange glow. Creates a "scanning" effect — like you're probing a circuit board with a multimeter. Uses canvas for performance.

TRACE CONNECTIONS

When cursor activates 2+ nearby nodes, faint orange lines draw between them — temporary circuit traces. Fade out over 0.5s when cursor moves away. Suggests live network connections forming.

WARM GLOW FOLLOWS CURSOR

The radial glow drifts toward cursor position with 800ms ease-out delay. Subtle — the glow's center lags behind the cursor, creating a "heat source" that follows attention. CSS transition only.

TAP RIPPLE (CLICK/MOBILE)

Click or tap anywhere in the section to emit a concentric ripple pulse — orange ring expanding outward + inner yellow ring. Two rings, different speeds. Like a sonar ping or network broadcast. DOM elements, auto-removed.

DATA FLOW LINES

Faint horizontal + vertical light traces animate along the grid lines. 7 horizontal (flowing right) + 5 vertical (flowing down). Different speeds and delays. Suggests data packets moving through infrastructure. CSS-only animation.

PRODUCTION NOTES

Canvas interaction → GSAP in production (already in stack). prefers-reduced-motion: disable data flows + node glow, keep static grid + stripes + edges. Mobile: touch events replace mousemove, tap ripple works natively.
