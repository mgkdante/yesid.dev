# Slice 11 — Navbar Redesign + 404 Page: Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the current fixed top-bar nav with a centered floating pill navbar (adaptive links per breakpoint, progress rail, metro dashboard menu overlay) and add a branded 404 "Station Under Construction" error page.

**Architecture:** The Nav component is rewritten in-place (`Nav.svelte`). The menu overlay is a new component (`MenuOverlay.svelte`) rendered by Nav. The 404 page is a new `+error.svelte` route with a `ConstructionScene.svelte` SVG component. All copy goes through `LocalizedString`. Existing wordmark SplitText animation is preserved. GSAP handles menu stagger and 404 typewriter animations.

**Tech Stack:** SvelteKit 2 + Svelte 5, TypeScript, GSAP (SplitText, gsap.timeline), Tailwind CSS + tokens.css, Vitest + @testing-library/svelte

**Design spec:** `docs/specs/2026-04-09-slice-11-nav-404-design.md`
