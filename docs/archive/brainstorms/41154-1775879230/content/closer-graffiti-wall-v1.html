3 — Full Mural

Giant "yesid." graffiti mural dominates the wall. Geometric shapes (circles, triangles), heavy drips, scattered tags ("DIGITAL INFRA", "BUILD RIGHT"), wheat-paste stickers. Board has an orange border glow. The wall IS the art — the board lives inside it.

Note: In implementation, the wall texture would use real SVG noise/grain, CSS patterns, and possibly a subtle concrete/brick image overlay at very low opacity. The graffiti "yesid." tag could use a custom SVG path for a hand-drawn feel. Drips would animate on scroll entrance with GSAP.
