Tech Stack — What Sells "Digital Infrastructure"?

Pick the stack that says "this guy can build anything." Retool removed. Select the ones that feel right.

1

PostgreSQL

Signal: "I know databases deeply." Universal respect. Every serious backend uses it.

2

TypeScript

Signal: "I write production-grade code." Industry standard. Shows you're not a script kiddie.

3

Python

Signal: "I do data, automation, and AI." Versatile. Every data pipeline and ML workflow uses it.

4

SvelteKit

Signal: "I build modern, fast frontends." Cutting-edge framework. The site itself IS the proof.

5

GSAP

Signal: "I create world-class animation." Rare skill. The hero animation IS the portfolio piece.

6

SQL Server

Signal: "I work in enterprise environments." Pairs with PostgreSQL to show database versatility.

7

Power BI

Signal: "I turn data into decisions." Enterprise dashboarding. Shows the full data→insight pipeline.

8

Docker

Signal: "I deploy and scale." Infrastructure-as-code. Shows you think beyond just writing code.

9

Node.js

Signal: "Full-stack JavaScript runtime." Pairs with TypeScript. Backend APIs, real-time, serverless.

Recommendation: Pick 5-6 max. Too many dilutes the message. Each one should earn its spot by proving a different capability.
