MOBILE: VERTICAL ACCORDION

On mobile (<768px), the diagram transforms into a vertical accordion. Each infrastructure layer becomes a collapsible section — tap to expand. Nodes within each layer wrap as flex items. Tapping a node opens a bottom sheet (slides up from bottom) instead of the sidebar. Filters become a horizontally scrollable pill bar. The Build Your Stack configurator goes full-width below the accordion.


Key interactions: Tap layer header to expand/collapse • Tap node for bottom sheet • Swipe down to dismiss • Prev/next arrows inside sheet • Horizontal scroll for filters
